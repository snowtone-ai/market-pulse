# /resume — セッション再開 Skill

## トリガー
長時間ギャップ後のセッション再開時、または「どこまでやったっけ？」と感じた時

## 実行手順

1. `.claude/state.md` を読む
2. `.claude/decisions.md` を読む
3. `.claude/issues.md` を読む（未解決エラーを確認）
4. 以下の形式でサマリーを出力する:

```
## セッション再開サマリー

### 完了済みタスク
- T-XXX: [タスク名]

### 進行中
- T-XXX: [タスク名]（[作業中のファイル]）

### 未着手（依存解消済み）
- T-XXX: [タスク名]（推奨：次のアクション）

### 未解決のエラー
- [issues.md から転記]

### 推奨：次のアクション
[具体的な1つのアクション]
```

5. @verifier を Task ツールで起動し、現在の実装の整合性を確認することを提案する
